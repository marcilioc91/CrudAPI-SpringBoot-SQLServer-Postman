## Descrição do Projeto:
O intuito desse projeto é de documentar meus estudos sobre Java.

Neste exemplo, estou criando um serviço de API para salvar informações de um CRUD básico de usuários, persistindo os dados no banco de dados SQL Server.
### Stacks utilizadas:
* Java 25
* Spring Boot
* Spring Web
* Spring Data JPA
* SQL Server
* Hibernate
* Maven

### Estrutura do projeto:
 ```src/main/java/com/exemplo/demo
 ├── controller
 │    └── UsuarioController.java
 ├── entity
 │    └── Usuario.java
 ├── repository
 │    └── UsuarioRepository.java
 ├── service
 │    └── UsuarioService.java
 └── DemoApplication.java
```
